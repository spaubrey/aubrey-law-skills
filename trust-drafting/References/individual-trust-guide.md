# Individual Revocable Trust — Drafting Guide

Load this reference when drafting the Revocable Trust for a single (individual) client.

Source template: `Assets/individual-revocable-trust.docx`

---

## Document Identity

- **File name:** `YYYY-MM-DD_[Trust-Name]_[CLIENT-LAST-NAME].docx`
- **Page size:** US Letter (12240 × 15840 DXA)
- **Margins:** Top 1080, Bottom 1080, Left 1008, Right 1008 DXA
- **Font:** Garamond throughout, 12pt body, 14pt bold titles
- **Color:** Black `000000` only — no navy, no teal, no gray
- **Header:** None — no logo, no header image
- **Footer:** Plain black text, document title and page number

---

## Opening Recital (Article One)

```
The date of this trust is _____________________, 20___.
The parties to this trust are [GRANTOR] (the Grantor) and [GRANTOR]
[and [INITIAL CO-TRUSTEE]] (collectively, the Trustee).
```

- If NO initial co-trustee: `The parties to this trust are [GRANTOR] (the Grantor) and [GRANTOR] (the Trustee).`
- If YES initial co-trustee: `The parties to this trust are [GRANTOR] (the Grantor) and [GRANTOR] and [INITIAL CO-TRUSTEE] (collectively, the Trustee).`

---

## Identifying the Trust (Section 1.01)

Trust identification clause — adjust trustee format:

- No co-trustee:
  `"[GRANTOR], as Trustee of the [TRUST NAME] dated _______________, 20___, and any amendments thereto."`

- With initial co-trustee:
  `"[GRANTOR] and [INITIAL CO-TRUSTEE], Trustees of the [TRUST NAME] dated _______________, 20___, and any amendments thereto."`

---

## Key Placeholder Substitutions

| Placeholder | Value |
|---|---|
| `[TRUST NAME]` | Full trust name from intake |
| `[GRANTOR]` | Client full name |
| `[INITIAL CO-TRUSTEE]` | Co-trustee name (if any); omit block if none |
| Date line | Leave as blank line or fill if date known |

---

## Trustee Succession (Article Three)

- **Section 3.02 — Trustee Succession while Grantor is Alive:**
  Insert primary successor trustee name and relationship.

- **Section 3.03 — Trustee Succession after Death:**
  Insert successor(s) in order. Use the names collected in intake.

- **Section 3.05 — Co-Trustee appointment:**
  Include if client named initial co-trustee. Otherwise retain section
  with blank as drafted in template.

---

## Family Information (Article Two)

Fill in:
- Spouse name (if married, even if no joint trust)
- Children names and DOBs (if minors)
- Note: "Descendants" references in the template are defined terms —
  do not alter.

---

## Pronoun and Number Rules

Individual trust = singular throughout:
- "I", "me", "my", "the Grantor", "the Trustee"
- Do NOT use "we", "our", "the Grantors", "the Trustees"

---

## Sections to Include / Exclude

| Section | Include? |
|---|---|
| Article Eight — Marital Trust | Include only if client is married |
| Guardian nomination | Include only if client has minor children |
| Article re: spouse | Adjust if client is single — remove spouse-specific distribution provisions |

**Important:** The individual trust template has provisions for a marital
trust and family trust. If the client is single (no spouse), these
sections may be present but will not be triggered at death. Review with
attorney if client is single — flag as `[REQUIRES ATTORNEY REVIEW —
client is single; verify Marital Trust provisions are appropriate]`.
